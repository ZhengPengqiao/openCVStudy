# openCVStudy

## 环境介绍

* 在Ubuntu 16.04平台下学习openCV的程序
* openCV 版本为 3.4.0

## 工程介绍

* 1_ShowImage
  * 使用opencv提供的函数,加载显示图片
* 2_erode
  * 腐蚀效果
* 3_blur
  * 使用均值滤波,对图像进行模糊效果
* 4_canny
  * 边沿检测
* 5_gray
  * 灰度效果
* 6_VideoCapture
  * 视频操作的使用(摄像头/文件,canny/normal)
* 7_Camshift
  * 彩色目标跟踪-根据鼠标款选区域的色度光谱来进行摄像头读入视频目标的跟踪
* 8_OpticalFlow
  * 光流(optical flow)法
* 9_lkdemo
  * 点追踪
* 10_objectDetection
  * 人脸识别
* 11_SupportVectorMachines
  * 支持向量机引导
* 12_SVMForNonLinearlySeparableData
  * 12_支持向量机之处理线性不可分数据
